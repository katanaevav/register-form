## Запуск проекта

1. Установить Node.js (использовалась v14.15.4)

2. Установить все зависимости.

```shell
npm install
```

3. Запустить.

```shell
npm run start
```

4. Браузер с проектом откроется автоматически по адресу
[http://localhost:3000/](http://localhost:3000/).
